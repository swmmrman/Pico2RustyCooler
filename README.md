# Pico2RustyCooler
A Pi Pico 2 based Evaporative Cooling system, with room monitoring

This software is not intended for use other then its original purose.  However, I am making it available so that other may learn or use it as a starting point.  

The intent of this project is to control a whole house fan, lights, and an evaportive cooling system.  This is to monitor and maintain conditions within a chinchilla breeding room.

Most portions will be run on RaspberryPi Pico 2 W(H)s.  Hardware will be listed as it is added.  I might even create diagrams and store them here.

No support will be offered.
